## Install

1. Clone this repository.
2. Run following command for build.  
    ```sh
    npm run build
    ```
3. Go to Extentions setting in Chrome.
4. Turn on Developer mode.
5. Press `Load unpacked`.
6. Select `dist/savitri` folder.
